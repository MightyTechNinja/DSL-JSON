GraphLogic Rules Engine DSL Documentation

This documentation provides an overview and instructions for writing code using a Domain-Specific Language (DSL) to define rules in a rules engine. The DSL support almost everything that the normal programming language supports including Varibles, Actions, Conditions and Loops.

Complete Example

Varibles

Syntax

VAR variable_name = value

Example:

VAR x = 10
VAR y = GetY()

Actions

Syntax

action_name()

Example:

doSomething()

Conditions

Main Syntax

IF condition THEN
    /* if content */
ELSE IF condition THEN
    /* else if content */
ELSE 
    /* else content */
ENDIF

Condition Syntax

Condition

(entity [RELATION] {property: value})

Operator

AND, OR, NOT

Relation Operator

(- ... ->)

Example:

IF (Person {first_name: "John}) THEN doSomething() ENDIF
IF (Person {first_name: "John}) AND (Skill {name: "JavaScript"}) THEN learnJavaScript() ENDIF
IF (Person {first_name: "John"}) - (Person HAS {years: 10}) -> (Skill {name: "JavaScript"}) THEN learnJavascript() ENDIF

Loops

Syntax

For Loop

FOR iterator = start TO
    /* content */
NEXT iterator

Foreach Loop

FOREACH iterator in collection
    /* content */
NEXT iterator

Example:

FOR i = 0 to 10
    doSomething()
NEXT i
FOR item in items
    doSomething()
NEXT item

Complete Example

VAR x = 10
VAR y = 20

FOR i = 0 TO 5
    IF (Person {first_name: "John"}) - {years: 10} -> (Skill {name: "JavaScript"}) THEN
        doSomething()
    ELSE
        VAR items = getItems()
        IF (Person CONTAINS {first_name: "John"}) AND (Person {age: 30}) OR NOT (Skill {name: "JavaScript"}) THEN
            doSomethingElse()
        ENDIF
    ENDIF

    FOR j = 0 TO 3
        IF (Users SUPPORT {email: "example@example.com"}) THEN
            doAnotherThing()
        ELSE IF (Users {email: "example2@example.com"}) THEN
            doAnotherThing2()
        ELSE
            VAR z = 5
            doYetAnotherThing()
        ENDIF
    NEXT j

    FOREACH item IN items
        IF (Users INHERITS_FROM {email: "example@example.com"}) THEN
            doAnotherThing()
        ELSE
            VAR z = 5
            doYetAnotherThing()
        ENDIF
    NEXT item
NEXT i
